# py-echo-bot
A echo bot written on python language :)

#installation
```
#Python installation

sudo apt-get install python2.7

sudo pip install pytelegrambotapi

#clone bot

git clone https://github.com/Mrhalix/py-echo-bot.git

#run bot

cd py-echo-bot

python bot.py

```
